# BIA PSA Tool v1.0

Budget Impact Analysis tool with Probabilistic Sensitivity Analysis.

## Usage

Open `index.html` directly in any modern browser — no server or build step required.

Or visit the live deployment at: **[TODO: add your URL after deploying]**

## Features

- Multi-comparator budget impact model
- Prevalence-based patient funnel with growth rates
- Monte Carlo PSA (10,000 iterations, Web Worker)
- One-way sensitivity / tornado diagram
- Scenario analysis (Low / Base / High)
- Threshold / break-even analysis
- Auto-saves to localStorage, JSON export/import
- Print to PDF

## Deployment

Hosted on GitHub Pages. Every push to `main` auto-deploys.
